# Quickstart

## Installation

Install Block Wrangler from your package manager of choice, or from source:
<Tabs>
<TabItem value="pip" label="pip">
```bash
pip install --user block-wrangler
```
</TabItem>
<TabItem value="pipx" label="pipx">
```bash
pipx install block-wrangler
```
</TabItem>
<TabItem value="from_source" label="from source">
```bash
cd some/folder
git clone https://github.com/camplowell/block_wrangler.git
cd block_wrangler
pip install --user .
```
> Note: installing from source will not generate a portable requirements.txt file, so be aware before publishing your shaderpack on Github.  
> However, it will allow you to create custom versions of Block Wrangler for your own use, or to contribute to the project.
</TabItem>
<TabItem value="venv" label="virtual environment">
```bash
python -m venv ./venv
source ./venv/bin/activate
pip install block-wrangler
```
</TabItem>
</Tabs>

<details>
<summary>Extra features</summary>

To get more detailed error messages when something goes wrong, add `[fuzzy_tags]` to your install command, like so:
```bash
pip install block-wrangler[fuzzy_tags]
```
</details>

## Usage

At the root of your shaderpack, create a `block_properties.py` file with the following contents:
```python title="block_properties.py"
from block_wrangler import *
from pathlib import Path


shaderpack_root = Path(__file__).parent

def main():
	tags = load_tags()

	mapping = BlockMapping.solve({
		'sway': tags['sway'],
		'sway_bottom': tags['sway/lower'] + tags['sway/short'], # Tags can be combined with the +, -, and & operators
		'crops': tags['minecraft:crops'], # Vanilla tags are included
		'water': blocks('minecraft:water') # Individual blocks can also be referenced by name
	})

	with shaderpack_root.joinpath('shaders/block.properties').open('w') as f:
		f.write(mapping.render_encoder())
	with shaderpack_root.joinpath('shaders/util/block_properties.glsl').open('w') as f:
		f.write(mapping.render_decoder())

if __name__ == '__main__':
	main()
```

run `python block_properties.py` to generate the block properties and decoder GLSL files.

Be sure to generate a requirements.txt file using `pip freeze > requirements.txt` to let contributors know what they need to build your shaderpack!
